# FileTree-SN2
SN2 mod that adds a FileTree so Support can help better understand the game folder structure .
1. download ue4ss and extract it to ```/home/cookay/.steam/steam/steamapps/common/Subnautica2/Subnautica2/Binaries/Win64/```
2. download Filetree and extract it to ```/home/cookay/.steam/steam/steamapps/common/Subnautica2/Subnautica2/Binaries/Win64/ue4ss/Mods```
3. start Subnautica 2
